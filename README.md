# 💰 AI Budget Calculator

A modern AI-inspired Budget Calculator web application built using React, Tailwind CSS, and Recharts.

This project helps users manage monthly expenses, track savings, visualize spending patterns, and store financial records locally.

---

# 🚀 Features

## ✅ Smart Budget Management

* Add monthly income
* Track food, travel, shopping & other expenses
* Automatic savings calculation
* AI-style financial suggestions

## 📈 Expense Analytics

* Interactive Pie Chart visualization
* Real-time expense tracking
* Monthly analytics dashboard

## 🌙 Dark Mode Support

* Toggle between Light & Dark theme
* Smooth UI transitions

## 💾 Local Storage Integration

* Automatically saves user data
* Keeps monthly reports even after page refresh

## 🤖 AI Ready Structure

* Prepared for future OpenAI API integration
* Expandable AI financial insights system

## 📱 Mobile Friendly

* Responsive modern UI
* Future-ready for React Native conversion

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-budget-calculator.git
```

## 2️⃣ Open Project

```bash
cd ai-budget-calculator
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start Development Server

```bash
npm run dev
```
